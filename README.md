# Prankster IOS App 
<div id="top"></div>
	
## Technologies
Project is created with:
* React
* React Native
* TypeScript
* React Navigation

## How to run

1. Clone the repo:

   ```sh
   git clone https://github.com/dmtrhrytsak/prankster-ios-app.git
   ```
2. Install packages:

   ```sh
   npm install
   ```
3. Run the app

   ```
   npm start
   ```
